Final Project: Tata Surya Interaktif

Haloo! Saya Alita. Website ini adalah puncak dari perjalanan belajar saya dalam program Final Project AI dan Programming SMA-Gap Year Brain Academy Online by Ruangguru.

Website ini adalah presentasi akhir saya. Tujuan utamanya adalah menciptakan simulasi Tata Surya yang realistis dan interaktif, dengan menggabungkan ilmu astronomi dan keterampilan Web Programming.

Detail Pengembangan Teknis

Proyek ini sepenuhnya dikembangkan menggunakan web standards dan library yang teruji:

  • HTML5: Saya menggunakan HTML untuk menyusun kerangka konten utama website.
  • Three.js (JavaScript Library): Ini adalah inti dari simulasi 3D. Saya menggunakan Three.js untuk merender grafis planet, mengatur tekstur, dan membuat efek cahaya Matahari.
  • Logika Program: Saya menulis kode JavaScript untuk mengontrol berbagai fungsi penting:
    • Pergerakan Planet: Mengatur kecepatan rotasi dan orbit planet agar mendekati data nyata.
    • Interaktivitas (Raycasting): Logika yang memungkinkan pengguna mengklik planet, dan kode akan merespons dengan memunculkan detail informasinya.
    • Planet, Matahari, Orbit, & Sabuk Asteroid: Membuat objek 3D kompleks ini menggunakan geometri dan material khusus Three.js.
